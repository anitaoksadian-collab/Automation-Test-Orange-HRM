# Automation Test - OrangeHRM

Repositori ini berisi skrip otomatisasi pengujian (*automation testing*) untuk aplikasi **OrangeHRM** menggunakan [Cypress](https://www.cypress.io/). 

Selain pengujian terotomatisasi, seluruh dokumentasi pengujian manual, skenario pengujian, serta pelaporan *bug* dikelola secara terpusat melalui tautan di bawah ini.
